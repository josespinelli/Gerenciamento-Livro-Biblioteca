Cenário: 3 - Cadastro de Livros

Este sistema foi desenvolvido para gerenciar uma biblioteca de livros, permitindo ao usuário cadastrar novos livros, visualizar todos os livros cadastrados e obter algumas estatísticas, como o livro mais antigo, a quantidade de livros publicados após 2020 e a média dos anos de publicação dos livros.

Assistente de apoio:
Para a criação deste sistema, utilizei o assistente Deepseek para auxiliar na estruturação do código e na correção de erros, como acessar arquivos e código para usar cores no terminal.

Descrição geral do que faz o sistema:
O sistema permite ao usuário:
1. Cadastrar novos livros, informando nome, autor e ano de publicação.
2. Visualizar todos os livros cadastrados, com informações como nome, autor e ano de publicação.
3. Obter estatísticas sobre os livros cadastrados, incluindo o livro mais antigo, a quantidade de livros publicados após 2020 e a média dos anos de publicação.

Descrição de como executar:
1. Certifique-se de ter o Python instalado em sua máquina.
2. Salve o código em um arquivo com extensão ".py", por exemplo, "main.py".
3. Execute o arquivo Python usando o comando "python main.py" no terminal.
4. Siga as instruções exibidas no menu para cadastrar livros, visualizar a biblioteca ou sair do sistema.
